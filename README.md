# Resume Site

This repository contains my personal resume site, built from the minimalist CV template by Bartosz Jarocki.

## About

The site is a clean, print-friendly resume built with a modern web stack and deployed as a static site. Content lives in a single data file so it is easy to keep up to date.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui (Radix UI)
- **GraphQL**: Apollo Server + TypeGraphQL
- **Package Manager**: pnpm
- **Deployment**: Optimized for Vercel

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.
